GitHub Sync
Sync your data across devices via a private GitHub Gist. Requires a GitHub Personal Access Token with gist scope.
Personal Access Token
Gist ID (auto-created on first push)
Setup guide:
1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Generate new token → Name: Lumo → check gist scope → Generate
3. Paste token above → tap Push & Sync
4. On another device: paste same token + the Gist ID that appears → tap Pull
